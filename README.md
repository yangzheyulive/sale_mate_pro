### SaleMate是AI对话式自动营销工具
> 私域运营现状：没有只是体系，不知道监控哪些数据；人工太花精力，用CRM又要配置更多资源；平台限制无法转化成私域；客户活跃度低，流失速度高于拉新速度等；SaleMate的出现很好的解决了上述问题。

## 目标
> 用户仅需对AI输入一句“帮我生成合适我们企业的客户运营策略”，接下的工作都交给平台策略和AI帮你完成。

### 主要功能
- 基于AI进行对话式业务操作
- 根据客户信息自动下发运营内容
- 根据标签自动生成营销内容
- A/B测试自动更新运营内容
- 精准回答个性化问题
- 更具人文关怀的回复
